Input Directory: Select a folder that contains subfolders for each station and sampling date. The subfolders names must have a date followed by an underscore and the station abreviation in capitals. E.g. '20200918_VAD'.
Subfolders Structure: The subfolders must contain the sensor output files as the come out and they will be detected automatically. If an empty file or no file is found for on sensor, it will be ignored and a warning will be issued. Here is the list of files that are detected for each sensor:
  • BP: file ending with '_BP.csv'
  • CO2atm: file ending with '_CO2ATM.LOG'
  • TURB: 'Cat.TXT' file within a 'TURB' subfolder
  • DO: 'Cat.TXT' file within a 'DO' subfolder
  • CDOM: 'Cat.TXT' file within a 'CDOM' subfolder
  • PAR1: file ending with '_PAR1.csv'
  • PAR2: file ending with '_PAR2.csv'
  • DEPTH: file ending with '_NDEPTH.csv'
  • pCO2: file ending with '_PCO2.csv'
Output Folder: Any folder that you want to use as output. Beware that all duplicated files will be overwritten. Possible output file names:
log.txt, warnings.txt, <station abreviation>_raw_hf.csv.
Station Selection: Select all the stations for which you want to combine data. All the stations with missing folders will be ignored and a warning will be issued.
Parameter Selection: Select the parameters to look for. If no data is found for a parameter, the column will be filled with 'NA' and some warnings will be issued.
Run Merging: To start the merge, fill all the inputs and press the 'RUN' button in the right top corner. The merge will start only if all the fields are filled.
