Input Folders: Select a folder that contains '.csv' files with eihter the measured 10 minutes data with gaps or the full modeled 10 minutes data.
Input Files: The file names must have at least the station abreviation in capital letters (e.g. VAD) and finish by '.csv'. They must contains exactly the following columns in whatever order:
Date, BPmbar, CDOMppb, CO2atmppm, ConduScm, DOmgL, DischargeLsec, PAR1Lux, PAR2Lux, TurbiNTU, WaterTempdegC, pCO2ppm
Output Folder: Any folder that you want to use as output. Beware that all duplicated files will be overwritten. Possible output file names:
log.txt, warnings.txt, 10min_data.csv, 6H_data.csv, 12H_data.csv, 24H_data.csv.
Station Selection: Select all the stations for which you want to convert data. All the stations with missing files or information will be ignored and a warning will be issued.
Data To Save: Select the data to be saved and computed. For the 6H, 12H, 24H data the new intervals will be computed which might take some time. Note that the 10min data file will always be created but saved only if the right option is selected.
Run Convertion: To start the convertion, fill all the inputs and press the 'RUN' button in the right top corner. The convertion will start only if all the fields are filled.
